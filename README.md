## Project setup

```bash
$ npm install
```

## Run DB and RMQ

```bash
$ docker compose up -d
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Check the Swagger Docs

Swagger docs **[DOCS](http://localhost:8000/docs)**.

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
